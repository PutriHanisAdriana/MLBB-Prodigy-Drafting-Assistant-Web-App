# MLBB Prodigy: Drafting Assistant Web App

> Draft Smarter. Play Stronger.

A browser-based strategic drafting tool designed for Mobile Legends: Bang Bang (MLBB) players. This web app provides tier list rankings, hero statistics, counter-pick guidance and a 7-Day Performance Trend graph to help players make informed drafting decisions.

## Features

- **Tier List** — View all 132 heroes ranked from S to D tier based on current Mythical Glory meta data, with role and lane filters
- **Hero Statistics** — Sortable table showing each hero's Win Rate, Pick Rate and Ban Rate with search and filter support
- **Counter-Pick Guidance** — Find the top 5 heroes that counter or are countered by any selected hero
- **7-Day Performance Trend** — Interactive line chart showing Win Rate, Ban Rate and Pick Rate trends over the past 7 days
- **Hero Detail Page** — Full hero profile including tier, stats, counter picks and performance trend

## Tech Stack

- HTML, CSS, JavaScript
- Static JSON-based data (no external API required)
- No login or installation needed — fully accessible via web browser

## How to Use

1. Open the web app URL in your browser
2. Use the **navigation bar** at the top to access Tier List, Statistics or Counter Pick pages
3. Apply **role, lane or rank filters** to narrow down heroes
4. Click on any **hero icon** to view the full Hero Detail Page
5. Use the **Counter Pick page** during draft phase to make informed pick and ban decisions
